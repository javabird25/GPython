Building¶

Requirements for building¶

  1. Visual Studio 2017 or newer

  2. CMake

  3. Python, preferably the latest version

Instructions¶

  1. Copy lua_launcher\pygmod_launcher to garrysmod\addons.

  2. Copy src\python\pygmod to garrysmod\pygmod.

  3. Open Developer Command Prompt for VS (most likely residing in your Start menu, folder Visual Studio), cd to src\cpp.

  4. Run:

    cmake -G "NMake Makefiles" && nmake

  5. Move gmsv_pygmod_win32.dll and gmcl_pygmod_win32.dll from src\cpp to garrysmod\lua\bin.

  6. Move pygmod.dll from src\cpp to Garry’s Mod’s root directory (where hl2.exe resides).

Final directory structure should looks like this:

...\SteamApps\GarrysMod\ ─┬─ hl2.exe
                          ├─ ...
                          ├─ pygmod.dll
                          ├─ garrysmod\ ─┬─ addons\ ─── pygmod_launcher\ ─┬─ addon.json
                          │              │                                └─ lua\ ───── init.lua
                          │              └─ lua\ ────── bin\ ─────────────┬─ gmsv_pygmod_win32.dll
                          │                                               └─ gmcl_pygmod_win32.dll
                          └─ pygmod\ ────── pygmod\ ──────────────────────┬─ __init__.py
                                                                          ├─ _error_notif.py
                                                                          └─ ...
