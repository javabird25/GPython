Running Unit tests¶

Requirements¶

  1. Docker Desktop for Windows

Instructions¶

Python tests¶

  1. Run in a command prompt from the repository root:

    docker build -f python_tests.Dockerfile .

C++ tests¶

  1. Run in a command prompt from the repository root:

    docker build -f cpp_tests.Dockerfile .
