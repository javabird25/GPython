chat - manipulating the chat¶

This module provides tools for manipulating the chat.

gmod.chat.print(*values, sep=' ')¶

Prints the values to the chat.

Values may contain gmod.color.Color objects to change the color of the following text. You can change the color multiple times.

Newline is added after the message, and there is no way to change that.

Lua similar: chat.AddText()
