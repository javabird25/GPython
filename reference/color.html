color - color classes¶

class gmod.color.Color(r, g, b, a)¶

The color class which is used in various client functions, such as functions in modules gmod.chat and gmod.draw.

Color constructor requires four color components: red, green, blue and alpha (transparency).

Lua similar: Color()
