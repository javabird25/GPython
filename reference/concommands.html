concommands - creating and calling console commands¶

This module provides tools for creating and executing console commands.

class gmod.concommands.ConCmd(name: str, callback)¶

The console command class.

The callback is a function which will be called when the console command will be executed. It must receive 2 arguments:

  • Player who executed this command
  • str of arguments separated by spaces.

Lua similar: concommand.Add()

callback¶

This concommand’s callback.

readd()¶

Adds this concommand again if it was previously removed.

remove()¶

Removes the console command, so it will be not available for executing.

Lua similar: concommand.Remove()

gmod.concommands.exists(name)¶

Returns True if the console command with the name name was registered by Lua concommand.Add() or by ConCmd class constructor.

gmod.concommands.run(cmd: Union[str, gmod.concommands.ConCmd], args: Union[str, Iterable[str]] = '') → None¶

Runs the console command.

Parameters:args – space-separated args or iterable of args strs.

Lua similar: RunConsoleCommand()
