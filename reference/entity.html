pygmod.entity - Scripted Entity creation¶

Provides the Entity class which allows creating scripted entities (a.k.a. SENTs) by the means of PyGmod.

class pygmod.entity.Entity¶

Base class for scripted entities. To create a scripted entity, simply inherit this class.

class MyEntity(Entity):
    def Initialize(self):
        ...
    ...

Entity classname will be generated automatically and will look something like MyAddon.python.__shared_autorun__.MyEntity. You can set a custom classname by passing it as a keyword argument to the class inheritance parameters:

class MyEntity(Entity, classname="attrs_myentity"):
    ...

You can set the entity fields (described here) by setting the class-level attributes:

class MyEntity(Entity):
    Type = "point"
    Category = "My Entities"
    PrintName = "My Entity"
    Author = "Me"
    ...

By default, Base is set to "base_entity" and Type is set to "anim". Also, there is a default version of Draw(), so you don’t have to write it by yourself:

def Draw(self, _):
    self._.DrawModel()

Note

To initialize your entity, you should use Initialize instead of __init__():

class MyEnt(Entity):
    def Initialize(self):
        print(f"Hello, world from entity #{self._.EntIndex()}")
