streams: I/O to Garry’s Mod¶

This module contains custom I/O classes which redirect I/O to Garry’s Mod.

class gmod.streams.GmodConsoleErr¶

Class which redirects the error output (stderr) to the Garry’s Mod console.

All messages are printed with the red color.

write(s)¶

Writes string s to Garry’s Mod console with MsgC Lua function with red color.

class gmod.streams.GmodConsoleOut¶

Class which redirects the standard output (stdout) to the Garry’s Mod console.

Messages from the client realm are printed with the yellow color and messages from the server realm are printed with the blue color, just like print, Msg and other Lua functions do.

write(s)¶

Writes string s to Garry’s Mod console with Msg Lua function.

gmod.streams.setup()¶

Sets sys.stdout to a new GmodConsoleOut instance and sys.stderr to a new GmodConsoleErr instance. Being called in redirectIO_toGmod() in main.cpp of the C++ module.
