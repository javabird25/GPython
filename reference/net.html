net: communication between Client and Server¶

This module provides tools for communication between client and server.

exception gmod.net.SizeError¶

Indicates that there are too many values. Raised by send() if more than 255 values are passed.

gmod.net.send(message_name, *values, addressee=None, lua_receiver=False)¶

Sends a net message to the opposite realm.

Parameters:
  • message_name (str) – The message name. Has to be registered with util.AddNetworkString() GLua function.
  • values (iterable) – Iterable of values to append to this message.
  • addressee (Player or iterable[Player] or None) – Message addressee. Ignored when sending to server, but required when sending from server.
  • lua_receiver (bool) – Whether this message is intended to be received by Lua code.
Raises:
  • ValueError – if the addressee is None when sending from server.
  • SizeError – if more than 255 values are passed and lua_receiver is False.
