player - player utilities¶

This module contains the Player class which wraps the Lua Player class.

class gmod.player.Player(lua_obj)¶

Class that wraps the Player Lua class.

gmod.player.get_by_userid(userid: int)¶

Returns the Player object which refers to the player with the user ID userid.

Lua similar: Player()

gmod.player.getlocal()¶

Returns the player object of the current client.

Note

getlocal() will raise NotReadyError if called before the InitPostEntity event.

Lua similar: LocalPlayer()
